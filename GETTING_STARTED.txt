================================================================================
  TROUBLESHOOTING
================================================================================

Port 3000 in use?
  lsof -ti:3000 | xargs kill -9

Module errors?
  rm -rf node_modules
  npm install

ngrok keeps disconnecting?
  → Upgrade to paid plan, or use Tailscale

Want to deploy?
  → See SETUP_AND_DEPLOYMENT.md

Need to change colors?
  → Edit tailwind.config.js colors section

Arabic text not RTL?
  → Clear browser cache, check LanguageContext
